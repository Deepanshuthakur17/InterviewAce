# InterviewAce

InterviewAce is a Next-Gen AI Mock Interview Platform built with Next.js, React, and Tailwind CSS.

## Features

- **AI-Powered Analysis**: Get instant, granular feedback on your technical accuracy and communication skills.
- **Role-Specific Libraries**: Practice curated tracks for Frontend, React, Next.js, JavaScript, and more.
- **Progress Analytics**: Track your growth over time and identify weak topics.
- **Smart Resume Parsing**: Upload your resume to auto-generate customized mock interview questions.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Database**: Prisma
- **Authentication**: NextAuth.js
- **Animations**: Framer Motion

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser
